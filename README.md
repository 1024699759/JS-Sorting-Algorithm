# 十大经典排序算法

排序算法是《数据结构与算法》中最基本的算法之一。用一张图概括：

![十大经典排序算法 概览截图](res/sort.png)


名词解释：

**n**：数据规模

**k**：“桶”的个数

**In-place**：占用常数内存，不占用额外内存

**Out-place**：占用额外内存

**稳定性**：排序后2个相等键值的顺序和排序之前它们的顺序相同
